# Contributing to Disk-I-O Performance Analyzer Auto Optimizer

Thank you for your interest in contributing!

We welcome beginners and experienced developers.

---

## Getting Started

1. Fork the repository
2. Clone your fork

```bash
git clone https://github.com/<your-username>/Disk-I-O-Performance-Analyzer-Auto-Optimizer.git
```

3. Create a new branch

```bash
git checkout -b feature/your-feature-name
```

4. Make your changes
5. Commit your work

```bash
git commit -m "Add feature description"
```

6. Push changes

```bash
git push origin feature/your-feature-name
```

7. Open a Pull Request

## Development Guidelines

- Write clean and readable code
- Use meaningful variable names
- Add comments when necessary
- Follow existing project structure
- Keep pull requests focused on one task

---

## Finding Work

Check:

- Issues labeled `good first issue`
- Issues labeled `help wanted`
- Open milestones

---

## Reporting Bugs

When reporting bugs include:

- Operating System
- Python Version
- Steps to reproduce
- Expected behavior
- Actual behavior

---

## Pull Request Checklist

Before submitting:

- Code runs successfully
- No unnecessary files added
- Documentation updated if needed
- Pull request description completed
